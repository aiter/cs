### 动态类型 & 静态类型
静态类型
运行之前，就检查类型，比如在java/c++/c/golang,在编译阶段就检查类型。

动态类型
运行期间，判断类型

```
       +----------- Translation -------------------->--+
       |                                               |
+--------------+                              +--------------+
| Source Code  |                              | machine Code |
+--------------+                              +--------------+
编译型 +                          Compiled                          Run-Time
解释型                                                          Interpreted
```

#### lettuce
> Advanced Java Redis client
> * 支持同步、异步(Java 8's CompletionStage interface 和 Reactive Streams)
> * 使用 [Netty]() 和redis服务端通信
> * 支持独立(standalone)的redis服务器
> * 支持通过SSL独立(standalone)的redis服务器
> * 支持哨兵(sentinel)的方式
> * 支持集群(cluster)的方式
